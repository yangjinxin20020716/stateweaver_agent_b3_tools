# Bid Requirements

The project requires a local document and data analysis assistant. The system must read local files, summarize requirements, analyze budget tables, and generate traceable reports.

Mandatory deliverables:

- A Markdown report with project requirements, budget summary, risks and staffing suggestions.
- A JSON summary containing total budget, expected cost and risk level.
- Evidence lines that show which local files support each major conclusion.

Constraints:

- The workflow should run locally.
- All generated outputs should be reproducible.
- The process should save intermediate traces for review.
